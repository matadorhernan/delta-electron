# Desktop App for Delta Mexico

#### Usage and Installation

To use this app just cd into the cloned repository and use these lines.

```bash
npm i
npm run electron:windows
npm run electron:linux
npm run electron:mac
```

#### Live Development

```bash
npm start
```
